Challenge HTML/CSS – Les 12 Travaux d’Hercule

Ce projet a été réalisé dans le cadre de ma formation chez **O'clock** afin de tester mes compétences globales en **HTML** et **CSS**.  
L’objectif était de compléter une page web inspirée des *12 travaux d’Hercule*, en suivant une série d’étapes techniques et créatives.

---

### Objectifs du challenge

- Corriger les liens entre fichiers HTML, CSS et images
- Intégrer une police personnalisée via Google Fonts
- Styliser le menu navigation (stylisation via Flexbox et différents sélecteurs CSS)
- Travailler la hiérarchie typographique (titres, sous-titres)
- Injecter du contenu structuré depuis un fichier Markdown (ajout d'un lien internet)
- Mettre en place un layout responsive avec articles en grille (CSS Grid)
- Personnaliser les boutons avec couleurs, bordures et effets
- Ajouter des transitions et effets de survol fluides
- Superposer une légende sur une image principale
- Fixer le menu latéral pour une navigation persistante
- Adapter l’affichage pour les petits écrans (media queries)
- Réorganiser la navigation mobile en mode horizontal

---

### Compétences mobilisées

- Structuration sémantique en HTML5
- Sélecteurs CSS avancés et pseudo-classes
- Responsive design avec media queries
- Intégration de polices et assets externes
- Gestion du positionnement (flexbox, float, fixed)
- Transitions CSS et interactions utilisateur
- Organisation de fichiers et arborescence projet
- Lecture et intégration de contenu Markdown

---

### Résultat

Le projet final est une page web complète, responsive, avec une navigation intuitive et des effets visuels soignés.  
Chaque étape m’a permis de consolider mes bases et d’approfondir des notions clés du développement front-end.

---

### En résumé

Ce challenge m’a permis de :
- Appliquer concrètement mes connaissances HTML/CSS
- Travailler avec des consignes détaillées et des assets fournis
- Documenter mon travail de manière claire et professionnelle
- Me préparer à des projets plus complexes en front-end
